# Hejin - Handmade Leather Bags and Accessories

A modern ecommerce website template built with Next.js for a handmade leather products company.

## Features

- **Responsive Design**: Mobile-first design using Tailwind CSS
- **Product Catalog**: Showcase handmade leather bags, wallets, and accessories
- **Pages**: Home, Shop, About, Contact
- **TypeScript**: Fully typed for better development experience
- **App Router**: Uses Next.js 14+ App Router for optimal performance

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - Next.js app router pages
- `components/` - Reusable React components
- `lib/` - Utility functions and data
- `public/` - Static assets (add product images here)

## Customization

- Replace placeholder images in `public/` with actual product photos
- Update product data in `lib/products.ts`
- Modify styling in components using Tailwind classes
- Add more pages or features as needed

## Build for Production

```bash
npm run build
npm start
```

## Deploy

This project can be deployed to Vercel, Netlify, or any platform supporting Next.js.

## Technologies Used

- Next.js 14+
- TypeScript
- Tailwind CSS
- React
